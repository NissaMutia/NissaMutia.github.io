Project 1

Sentiment Analysis for Skincare Review Videos using Random Forest Classification

This project analysist the youtube skincare video into 5-stars system. Videos that used in this project limited to videos that contain only one product of one brand in the video. The result will be the name of the product and the overall review from the beauty vlogger.

Things that I do in this project is :

  • Collecting videos about skincare reviews
  • Converting videos into text using Google-Speech-Recognition
  • Labelling the text
  • Train the Random Forest Classifier
  • The accuracy result is up to 50%
